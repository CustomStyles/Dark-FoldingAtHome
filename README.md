# Dark Folding@Home
A dark theme for Folding@Home

## Installation
Install [Stylus](https://add0n.com/stylus.html) if you are using Firefox, Chrome or Opera.
If you are on Safari, install [Cascadea](https://cascadea.app/).<br/>
